🎮 Quick Start Guide
1)Install Pygame: Open a terminal (CMD) and run: pip install pygame.
2)Launch the Program: Run the launch_simple.py file. This script automatically opens both the Dashboard and the Sport Display.
3)Select Window Size: In the console that appears, press 1 and then ENTER to select the Standard size (recommended).
4)Vehicle Controls: * SPACE: Rev the engine in Neutral (N).ENTER: Accelerate in Drive (D) to see live Power and Torque.SHIFT: Apply the Brake to decelerate.

🎮 CONTROLS:

   SPACE       → Rev engine in Neutral (N)
                 Throttle load builds up (0-30%)
                 Power/Torque NOT displayed
                 
   ENTER       → Accelerate in Drive (D)
                 Power and Torque LIVE on right screen
                 Progress bars fill with acceleration
                 
   SHIFT       → Brake / Decelerate
                 Smooth deceleration
                 Gear downshifts automatically

═════════════════════════════════════════════════════════════════════════════════

📊 WHAT YOU'LL SEE:

   LEFT SCREEN (Dashboard - Redline_Revving_Sim):
   ├─ Tachometer (0-8000 RPM) - Top left
   ├─ Speedometer (0-280 km/h) - Top right
   │   └─ Progress bar fills with dark blue during acceleration
   ├─ Gear indicator - Bottom
   └─ Status: Neutral/Drive mode

   RIGHT SCREEN (Sport Telemetry - iDrive 7):
   ├─ Power (Putere) - 0-550 CP - Left side
   ├─ Torque (Cuplu) - 0-650 Nm - Right side
   ├─ Both update ONLY during acceleration in Drive mode
   └─ Real-time data with smooth animations

🔧 TROUBLESHOOTING:

   If nothing happens:
   └─ Try running LAUNCH.bat from the desktop or double-click it
   
   If windows are too small/large:
   └─ Run LAUNCH.bat again and choose different size (2, 3, or 4)
   
   If programs crash:
   └─ Make sure Pygame is installed: pip install pygame
   └─ Restart your computer and try again
